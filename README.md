📔 My Diary App (나의 일기장 앱)
컴퓨터공학 전공 프로젝트의 일환으로 제작 중인 Full-Stack 일기 관리 애플리케이션입니다.

사용자의 소중한 일상을 기록하고, 안전하게 저장하며, 직관적인 UI를 통해 관리하는 것을 목표로 합니다.

🚀 기술 스택 (Tech Stack)
Frontend

React: 컴포넌트 기반 UI 라이브러리

Vite: 빠르고 현대적인 프론트엔드 빌드 도구

Axios: 서버와의 비동기 통신을 위한 HTTP 클라이언트

Backend

Node.js & Express: 안정적인 서버 환경 구축

MongoDB Atlas: 클라우드 NoSQL 데이터베이스

Mongoose: MongoDB 객체 모델링 도구

✨ 주요 기능 (Key Features)
일기 작성 (Create): 제목과 내용을 입력하여 데이터베이스에 실시간 저장

일기 목록 조회 (Read): 저장된 일기 목록을 서버로부터 불러와 화면에 출력

보안 강화: .env 환경 변수를 활용하여 데이터베이스 접속 정보 및 민감한 API 키 유출 방지

컴포넌트 기반 설계: DiaryForm, DiaryList 등 독립적인 컴포넌트로 분리하여 유지보수성 향상
