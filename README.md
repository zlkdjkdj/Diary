📔 Diary-App (나의 소중한 기록, 일기장 프로젝트)
1. 프로젝트 설명

이 프로젝트는 바쁜 일상 속에서 흘러가 버리기 쉬운 소중한 순간들을 기록하기 위해 제작된 개인용 웹 일기장입니다.

단순한 텍스트 기록을 넘어, 그날의 감정과 생각들을 차곡차곡 쌓아두는 나만의 디지털 서재를 지향합니다. 사용자는 언제 어디서든 자신의 일기를 작성하고, 지나간 기록들을 한눈에 모아보며 과거의 나를 돌아볼 수 있습니다.

2. 주제 선택 이유

컴퓨터공학을 전공하며 다양한 데이터를 다뤄왔지만, 정작 나 자신의 일상을 기록하는 일에는 소홀했다는 것을 깨달았습니다.

"기록은 기억을 지배한다"는 말처럼, 매일의 짧은 단상이라도 기록해 두는 것이 개인의 성장에 큰 자산이 될 것이라 생각했습니다. 누구나 쉽게 접근하여 자신의 이야기를 남길 수 있는 직관적인 서비스를 직접 만들어 보고자 이 주제를 선택하게 되었습니다.

3. 주요 기능 및 서비스 내용

[기능 구성]

간편한 일기 작성: 복잡한 절차 없이 제목과 내용만으로 그날의 기록을 즉시 남길 수 있습니다.

실시간 기록 보관: 작성한 일기는 즉시 저장되어 데이터베이스에 안전하게 보관됩니다.

타임라인 목록: 내가 쓴 일기들을 날짜순으로 정렬하여, 마치 한 권의 책을 읽듯 과거의 기록을 훑어볼 수 있습니다.

직관적인 UI/UX: 사용자가 글쓰기에만 집중할 수 있도록 불필요한 요소를 최소화한 깔끔한 디자인을 제공합니다.
